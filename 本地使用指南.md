## 🍎 macOS 详细步骤（Clash Verge Rev）

### 1. 下载安装

```bash
# 使用 Homebrew（推荐）
brew install --cask clash-verge-rev

# 或从 GitHub 下载
# https://github.com/clash-verge-rev/clash-verge-rev/releases
```

### 2. 导入配置

**方式1：导入 YAML 文件**
1. 打开 Clash Verge Rev
2. 点击左侧"配置"
3. 点击"导入"或"+"按钮
4. 选择 `client-configs/clash-config.yaml` 文件

**方式2：使用 VMess 链接**
1. 打开 Clash Verge Rev
2. 点击"代理"标签
3. 点击"添加"或"导入"
4. 粘贴 VMess 链接

### 3. 启动代理

1. 在"代理"标签中选择 `VMess-ai.bless.top`
2. 点击右上角的"系统代理"开关（或菜单栏图标）
3. 选择"系统代理"模式

### 4. 测试连接

打开浏览器访问：
- https://www.google.com
- https://www.youtube.com

或使用命令行测试：

```bash
# 测试代理
curl --proxy http://127.0.0.1:7890 https://www.google.com

# 查看IP地址
curl --proxy http://127.0.0.1:7890 https://ip.sb
```

## 🪟 Windows 详细步骤（v2rayN）

### 1. 下载安装

1. 访问：https://github.com/2dust/v2rayN/releases
2. 下载 `v2rayN-Core.zip`
3. 解压到任意目录（如 `C:\v2rayN`）
4. 运行 `v2rayN.exe`

### 2. 导入配置

**最简单方式**：
1. 复制 VMess 链接（见上方）
2. 在 v2rayN 中点击：`服务器` → `从剪贴板导入批量URL`
3. 节点会自动添加

### 3. 启动代理

1. 右键系统托盘图标
2. 选择 `Http代理` → `开启Http代理` → `自动配置系统代理`
3. 选择刚添加的服务器节点

### iOS

1. 安装 Shadowrocket（App Store，需付费）
2. 打开应用，点击右上角"+" → "添加节点"
3. 选择"VMess"
4. 填写服务器信息（见上方"服务器信息"）
5. 保存并启用

## 🔧 本地代理端口

使用代理客户端后，本地会开启以下端口：

- **Clash**: 
  - HTTP 代理：`127.0.0.1:7890`
  - SOCKS5 代理：`127.0.0.1:7891`

- **V2Ray**:
  - SOCKS5 代理：`127.0.0.1:1080`
  - HTTP 代理：`127.0.0.1:1081`

### 手动配置应用使用代理

如果某个应用不支持系统代理，可以手动配置：

```
代理类型: HTTP 或 SOCKS5
代理地址: 127.0.0.1
代理端口: 7890 (Clash) 或 1080 (V2Ray)
```

## 🐛 常见问题

### 1. 无法连接

- ✅ 检查客户端是否已启动
- ✅ 检查是否选择了正确的节点
- ✅ 检查系统代理是否已启用
- ✅ 检查防火墙设置

### 2. 连接慢

- ✅ 尝试切换节点（如果有多个）
- ✅ 检查网络连接
- ✅ 重启客户端

### 3. 某些网站无法访问

- ✅ 检查代理规则设置
- ✅ 尝试切换为"全局代理"模式
- ✅ 检查网站是否被规则屏蔽
